### Q 1. What Is a Prompt?

A prompt is a structured instruction given to an AI model to get a desired output.

A good prompt is:

- Clear
- Specific
- Context-aware
- Goal-oriented
- Constraint-defined

> I define role, context, task, constraints, and expected output format. Clear, structured prompts produce precise and high-quality results.

### Q 2. How to Write a Powerful Prompt (Senior-Level Thinking)?

Use this formula:

- **Role:** Act as a senior frontend architect‚Ä¶
- **Context:** I am preparing for a senior React interview with 9 years experience.
- **Task:** Generate 20 advanced React questions with detailed answers.
- **Constraints:** Define format, length, tone, or structure.
  - ‚ÄúProvide short interview-ready answers.‚Äù
  - ‚ÄúInclude code examples.‚Äù
  - ‚ÄúKeep explanation under 100 words.‚Äù
  - ‚ÄúExplain with real-world use cases.‚Äù
- **Output Format:** Specify structure.
  - Provide: Question, Deep Explanation, Short Answer, Code Example

### Q 3. Strong Prompt Example?

```

real-world use cases, performance implications, and provide a short 3-line
interview-ready answer.

```

```
- Conceptual explanation in human-readable language
- Real-world frontend examples
- Common pitfalls and performance considerations

```
